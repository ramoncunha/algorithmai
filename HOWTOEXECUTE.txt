-- English

To run the code you must have installed Jupyter Notebook. Download it from http://jupyter.org/install
With Jupyter installed, open the file .ipynb and click Run. Fill the required fields.

If you prefer run the code manually in terminal, need to import the follow libraries
*random import randint
*math import sqrt

And don't forget to import the sucessor function, menor_caminho and busca_gulosa.
